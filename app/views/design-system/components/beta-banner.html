Beta banner

Identifies services that are in the Beta phase and allows users to provide feedback about the service.

We must measure success of services and provide a means for users to submit feedback about a service, in line with the Service manual.

For services which are officially live, use the Feedback banner instead.

When to use this component

Use when a service is in the Beta phase to allow users to provide feedback about the service.

When not to use this component

Don't use this component on services which are officially live and out of the beta phase, use the Feedback banner instead.

How it works

The beta banner is used on all beta services and appears at the top of all pages before the header.

The component contains a link to the service feedback page.

There has not been any specific research activity on this component.
