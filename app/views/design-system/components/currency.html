When to use this component

Use the currency input component when you need users to tell you an amount of money in a particular currency, for example pounds sterling, euros or dollars.

How it works

The component uses type="text" rather than type="number" to ensure that all users can enter the decimal symbol. And the pattern attribute is used to trigger a numeric keypad on iOS devices.

Use HTML entity values for the currency symbols:

Currency symbols
Symbol HTML Entity
£ &pound;
$ &dollar;
€ &euro;

The input does not prevent users from typing invalid numbers because that makes the interface seem unresponsive. Instead rely on clear labels, hints and error messages.

The currency symbol is styled differently to the input so that users don’t confuse it with the input’s value.

Research on this component

We tested this component in the Alpha phase of the public register service. The component was used in a service for the legal team to add and manage settlements.

Where this component is used

There are no live services using this component.
