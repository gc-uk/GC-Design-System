Cards

Cards are used to display information to users which may include additional functionality such as links or buttons.

Linked cards

Linked cards provide users with information and link to more details relating to that card.

Multiple cards can be grouped together and when this is done, all cards will match the hight of the tallest card.

This card should not be used where multiple options may exist for the card's action. Instead, use the linked list card.

These cards must be positioned using the layout classes, for example: govuk-grid-column-one-third.

Linked list cards

Linked list cards provide users with information and a list of links to actions relating to the card.

These cards must be positioned using the layout classes, for example: govuk-grid-column-one-third.
