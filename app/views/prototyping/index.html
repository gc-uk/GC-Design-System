Prototyping

This guide helps you to create prototypes of Gambling Commission services using the Design System

Before you start

You'll need to request the following software to be installed through an IT Helpdesk request, before you can build and run any prototypes.

  • Install Node.js LTS version
  • Install Git Bash
  • Install VSCode

If you want to publish prototypes, you'll also need to:

  • Create an account on GitHub
  • Create an account on Heroku

Downloading the kit

Download the latest release of the GC Prototype Kit (link opens in a new window) from the GitHub repository.

Unzip the release into a folder on your C drive. Do not extract to a folder which is synced with OneDrive or Sharepoint.

Install and run the kit

Open the folder containing the extracted files in VS Code.

Check the VSCode terminal is set to bash.

In the terminal run:

npm install

When the install has finished, run:

npm start

Follow the link to the URL provided in the terminal, you should see the prototype kit homepage.

Quit the kit

In the terminal, press CTRL+C together, this will stop the kit running.

Restarting the kit

npm start
